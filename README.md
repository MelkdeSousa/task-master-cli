## Requirements

- Python 3
- Pip
- Pipenv

## Get Started

- Clone repository

  ```bash
  git clone https://github.com/MelkdeSousa/task-master-cli.git
  ```

- Create virtualenv with pipenv

  ```bash
  pipenv --three
  ```

- Install dependencies

  ```bash
  pipenv install
  ```

- Build

  ```bash
  pipenv install -e .
  ```

- Run

  ```bash
  task-master --help
  ```

## Features

- Add task
- Complete task
- List tasks
- Next tasks
- Delete task
